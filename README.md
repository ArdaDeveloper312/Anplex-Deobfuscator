# Anplex 🟠 — Java Deobfuscator AI

Anplex is an **agentic Java deobfuscator AI**. It takes an obfuscated
`.jar`/`.class`, detects the obfuscator (ZKM, JNT, JNIC, ProGuard, Stringer,
Allatori …) and works the code in an **autonomous loop**: decrypts strings,
folds opaque predicates & junk flow, renames identifiers to readable names —
**iteratively, until nothing changes anymore (fixed point)**. The result: a
deobfuscated JAR + decompiled source.

# How to start the Java Deobfuscator?

1. Extract the .zip file
2. type cmd in the folder
3. type exact this command: py server.py [YOU NEED TO HAVE PYTHON INSTALLED AND JDK]
4. Enjoy the JNT, Proguard, Zelix Class Master Auto deobfuscation
